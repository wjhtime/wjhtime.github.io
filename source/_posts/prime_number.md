求n以内的质数

#### 思路
```
质数是指大于1的正数，只能被1和本身整除
简洁方法：如果到n/2还没有查找到，则该数为质数
1. 从2开始，循环小于n的所有数
2. 嵌套循环，循环小于每个值的二分之一的数
3. 外层对内层数取余，如果等于0，则不是质数，跳出循环
4. 内层循环结束，判断值是否超过了i/2，如果是，加入到结果数组
```

#### 实现
```php
function find_prime($n) {
    $result = [];
    for ($i=2; $i<=$n; $i++){
        for ($j=2; $j<=$i/2; $j++){
            if ($i % $j == 0){
                break;
            }
        }
        if ($j > $i/2){
            array_push($result, $i);
        }
    }
    return $result;
}
```
